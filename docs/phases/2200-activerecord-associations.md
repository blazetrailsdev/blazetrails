# Phase 2200: ActiveRecord Associations

## Goal

Implement the association macros so models can declare relationships.

## Scope

- `belongs_to`
- `has_one`
- `has_many`
- `has_many :through`
- `has_one :through`
- `has_and_belongs_to_many`
- Association options: `foreign_key`, `class_name`, `dependent`, `inverse_of`
- Eager loading: `includes`, `preload`, `eager_load`
- Building and creating through associations
